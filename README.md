# ZERTZ (Web)

Браузерная реализация настольной игры ZERTZ с поддержкой локального (hot-seat) и онлайн (мультиплеер через сервер) режимов.

## 🛠 Технологический стек
- **Frontend:** React, TypeScript, Vite, TailwindCSS
- **State Management:** Zustand (`gameStore` для локальных игр, `roomStore` для онлайн)
- **База данных:** IndexedDB (ранее), PostgreSQL (бекенд)
- **Backend:** Express.js (Node), pg (PostgreSQL)
- **Рендеринг:** SVG доска с координатной сеткой (axial coordinates)

## 🗺️ Игровые механики
Игра полностью реализует правила оригинального ZERTZ (включая постановку, взятие, изоляции и проверку свободных колец).
Размеры досок: 37, 48 и 61 кольцо.
Также доступна система рейтинга Глико-2, списки игроков, глобальный чат и вызовы.

## 🚀 Запуск проекта
Для локальной разработки:
```bash
npm install
npm run dev
```

Для сборки проекта в production:
```bash
npm run build
```

## 🔄 План рефакторинга
В данный момент проект находится в фазе архитектурного рефакторинга.
Основные шаги (цель: гибкие, лёгкие функции):
1. **Декомпозиция монолитов:** Разделение компонента `MainMenu.tsx` (>1000 строк) на небольшие UI-компоненты (модальные окна, панели).
2. **Локализация:** Дробление огромного объекта `i18n.ts` на отдельные JSON-словари или файлы по языкам (`ru.ts`, `en.ts`).
3. **Выделение хуков:** Перемещение логики матчей, онлайн-вызовов и парсинга URL в отдельные пользовательские хуки (custom hooks).
